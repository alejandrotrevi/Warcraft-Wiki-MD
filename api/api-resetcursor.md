Resets mouse cursor.

 ResetCursor()

==Arguments==
: ''none''

==Returns==
: ''none''

==Details==
: Function resets mouse cursor into its default shape, if it has been previously altered by [[API SetCursor|SetCursor(cursor)]]. Calling <code>ResetCursor()</code> is equivalent to calling <code>SetCursor(nil)</code>.
